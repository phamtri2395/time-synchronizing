# Time Synchronizing
Synchronize time from time.gov and display data in human readable format to screen.

## Screenshots

## How to run
1. Clone this project.
2. Go to project folder & run ```npm install```.
3. Run ```npm start```.
4. Now server is on http://localhost:3000.
